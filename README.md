# Low Power

On Deek-Robot board:
* Cut the power led
* !!! Do not cut the regulator !!! On deek-robot board, need to unsolder the regulator Vout instead

[Details here] (http://forum.mysensors.org/topic/230/power-conservation-with-battery-powered-sensors)
